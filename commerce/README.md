# Commerce (Auctions)

## Overview
An eBay-like e-commerce auction site where users can post auction listings, place bids on listings, comment on those listings, and add listings to a "watchlist".

## Features
- **Auctions**: Create, view, and bid on listings.
- **Watchlist**: Users can track active listings.
- **Categories**: Filter listings by category.
- **Comments**: Discuss listings.
