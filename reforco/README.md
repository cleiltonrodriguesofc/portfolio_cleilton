## Overview
This is a SaaS application for private teachers to manage student attendance, payments, and communication using a multi-tenant architecture.

## Features
- **Student Management**: detailed profiles and academic history.
- **Attendance Tracking**: Digital call lists and presence reports.
- **Financial Management**: Track payments and generate monthly reports.
- **Communication**: Automated WhatsApp billing and status messages.

## Key Components
- `models.py`: Defines Student, Payment, and Attendance schemas.
- `views.py`: Business logic for the management dashboard.
- `templates/`: User interfaces for teachers and administrators.
