# Mail

## Overview
A front-end email client that makes API calls to send and receive emails, functioning as a Single Page Application (SPA).

## Features
- **Inbox/Sent/Archive**: Categorized email views.
- **Compose**: Send emails to other users.
- **SPA Experience**: No page reloads for navigation.
- **Read/Unread Status**: Tracks email state.
