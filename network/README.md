# Network

## Overview
A Twitter-like social network website where users can write posts, follow other users, and "like" posts.

## Features
- **Feed**: View posts from all users or only followed users.
- **Posting**: Create new text-based posts.
- **Interactions**: Like/Unlike posts asynchronously.
- **Profiles**: View user profiles and follower counts.
- **Pagination**: Navigate through pages of posts.
